# pTens-v2.0 项目概要文档

## 1. 项目概述

**项目名称**: pTens-v2.0  
**硬件平台**: GD32F30x (ARM Cortex-M3)  
**开发工具**: MDK-Keil  
**项目类型**: 嵌入式固件 (TENS经皮电神经刺激设备)

TENS (Transcutaneous Electrical Nerve Stimulation) 是一种通过皮肤表面电极刺激神经的物理治疗设备，本项目实现了一款多通道TENS治疗仪的固件。

## 2. 系统架构

### 2.1 主要功能模块

| 模块 | 功能描述 |
|------|----------|
| **TENS核心** | 主状态机管理、处方数据加载、输出控制 |
| **HV高压输出** | 0-30级强度调节，控制刺激电压 |
| **疗法管理** | 存储/加载10种疗法处方数据 |
| **PWM输出** | 生成TENS波形信号 |
| **按键处理** | 电源键、确认键、强度+/-、设置键 |
| **显示屏** | LCD显示界面 (GUI) |
| **蓝牙通信** | 串口协议与手机APP通信 |
| **电池管理** | 电量检测、充电状态监测 |
| **振动/加热** | 辅助按摩和热疗功能 |
| **EEPROM模拟** | 内部Flash模拟EEPROM存储 |

### 2.2 状态机

```
TENS_NONE (初始) -> TENS_IDLE (待机) -> TENS_RUN (运行)
                                              |
                   TENS_PAUSE (暂停) <--------+
```

## 3. 通信协议

### 3.1 蓝牙通信协议 (TensProtocol)

通过USART1串口与外部设备通信，支持以下命令:

- **系统命令**: 握手、获取信息、版本查询、心跳、固件升级
- **控制命令**: 输出时间、控制启停、强度设置、通道设置  
- **数据命令**: 疗法数据下载

## 4. 硬件资源

### 4.1 外设使用

- **TIMER0**: PWM输出
- **TIMER2**: 马达提示音
- **USART1**: 蓝牙串口通信
- **DMA0**: PWM DMA传输
- **GPIO**: 按键、LCD、HC595移位寄存器、继电器等

### 4.2 Flash存储布局

| 地址 | 用途 |
|------|------|
| 0x08001800 | APP程序起始 |
| 0x08010800 | 固件升级区 |
| 0x08027000 | 设备信息存储 |
| 0x0802F000 | 疗法处方数据 |

## 5. 关键参数

- **默认治疗时间**: 1800秒 (30分钟)
- **TENS强度等级**: 0-30级
- **加热等级**: 0-4级 (30°C/40°C/50°C/60°C)
- **振动模式**: 0-5模式
- **处方数据数量**: 10个
- **最大处方数据包**: 1100字节

├── battery.c/h       - 电池管理
├── heat.c/h          - 加热控制
├── vib.c/h           - 振动控制
├── hc595.c/h         - 移位寄存器驱动
